## SQL – Ejemplos prácticos

Esta carpeta contiene ejemplos de SQL usados en escenarios reales de ingeniería de datos:

- Dedupe con `ROW_NUMBER()` y claves compuestas.
- Joins entre múltiples tablas.
- `MERGE` para tablas históricas/transaccionales.
- Cálculos con `CASE WHEN` para estatus y categorías.

Cada archivo `.sql` es un ejemplo independiente, con comentarios para entender la lógica.
